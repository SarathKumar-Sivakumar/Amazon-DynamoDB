# Amazon-DynamoDB

1 Contains  zip file of the project (webapp.zip) 
2 Contains  war file of the project(webapp.war)
3 Contains readme file

Steps to do:
1 Create aws account
2 Create dynamodb table for putting the data
3 Create s3 bucket for  uploading the images
4 Install aws toolkit for java in eclpise
5 Run the project on AWS elastic beanstalk.
6 Use the below link for running the project.
http://cloudenvir-kcg2b4qcqy.elasticbeanstalk.com/index.jsp
